NOTICE

The files of this directory is an example of debugging C/C++ by VS Code.
The following 2 files need to be updated according to your project.
1. launch.json
    a. "program", Program that will be debuged.
    b. "args", Command line arguments the passed to the program.
    C. "cwd", The working directory of target.
    d. "envFile", The environment script.

2. env
    Key value pairs that separated by equals sign per line. E.g.
    KEY=VALUE
